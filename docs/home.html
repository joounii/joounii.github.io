Hi, I’m Joounii, a 18 years old, passionate software developer with a diverse background in various programming languages and technologies. My journey in coding started back in 7th grade, where I was first introduced to programming through Scratch, a beginner-friendly platform. From there, my interest in development quickly grew when a friend introduced me to Unity where I began learning the fundamentals of game development. Though I started with just the basics, it set the foundation for my future in coding.

After diving into HTML, CSS, and a bit of JavaScript on my own, I took a major step forward in my career by joining Open Circle after 9th grade. I joined the company as an apprentice software developer. During my apprenticeship, I gained hands-on experience in full-stack web development. I became proficient in Ruby on Rails for both backend and frontend development. In school, which I attended two days a week alongside my apprenticeship, I also picked up Java fundamentals. Additionally, I learned Bash scripting there, and one of my more interesting projects was a console-based game called 20 Wins, which was created entirely with Bash.

My passion for learning didn’t stop at work or school. In my free time, I taught myself Python, which led to several exciting personal projects. With Python, I developed:

  • A tool that can create an executable from any website
  • A Perlin Noise Generator for creating natural-looking random patterns
  • A Discord bot with some fun commands

Unfortunately, after three years at Open Circle, I had to pause my apprenticeship due to health reasons. However, that hasn't slowed down my progress or passion for coding. Currently, I’m working on two exciting projects:

  • Multiplayer Pong Game – This is a self-directed project aimed at teaching myself how multiplayer games work, how they communicate with servers, and how to implement the TCP protocol.
  • Block Dodge – A mobile game built with Unity and C#. I’m aiming to release the first version of the game within the next six months.

All my public projects, except for Block Dodge, can be found on my GitHub account. Through every project and challenge, I continue to grow and improve as a developer, driven by a deep passion for coding and problem-solving.
